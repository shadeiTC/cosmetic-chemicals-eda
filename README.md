# cosmetic-chemicals-eda
Exploratory Data Analysis of cosmetic chemicals data to identify trends, chemical usage patterns, and potential insights using Python, Pandas, Matplotlib, and Seaborn

## Objective
The goal of this project is to explore how different cosmetic chemicals appear and change over time in reported data, and to visualize key trends using data analysis techniques.

## Dataset
The dataset contains cosmetic chemical reports from 2009 to 2020, including information such as:
- Year of report
- Chemical names
- Frequency of occurrence
- reporting across brands

## Tools and Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Steps in Analysis
- Data cleaning and filtering
- Handling missing values
- Grouping data by year and chemical name
- Identification of the most frequently reported chemicals
- Creating trend analysis tables
- Visualizing trends using bar charts, line charts
  

## Key Findings
- Chemical usage varies significantly across years
- Some chemicals show consistent reporting trends over time
- Visualizations help highlight changes in chemical frequency across the dataset
- Analyzed the top 5 chemical occurrences across product categories

## How to Run This Project
1. Clone or download the repository
2. Ensure `cosmetics_final_EDA.csv` is in the same folder as the notebook
3. Open the Jupyter Notebook (`.ipynb` file)
4. Run all cells to reproduce the analysis
